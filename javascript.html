Javascript in a Webpage

Since I'm planning on including image carousels on this very website, the one above you is carousel-ception! Isn't that cool!? An carousel with images of carousels.

Anyways, enough blabbering about that. The implementation was very different from writing things in css and html, so it was a bit tricky for me even though it was just copying and pasting. I was more used to writing things out and messing with css. Just like everything else I've learned so far, though, I'll improve more and more so long as I keep practicing similar things.
